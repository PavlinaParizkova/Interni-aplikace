# 🧠 ai-prompt-library-processes

Vítejte v centrální knihovně AI agentů a procesů pro marketingový tým **AIR TEAM**. Tento repozitář slouží jako kontextový mozek pro editor Cursor.

## 🚀 Jak používat tuto knihovnu v Cursoru
Při práci v Cursoru můžete vyvolat konkrétní expertní roli pomocí symbolu `@`:

* **Pro tvorbu obsahu:** Použijte `@Webova-stranka-UX.md` nebo `@PPC-zadani.md`.
* **Pro operativu:** Použijte `@ClickUp-task.md` pro správné naformátování úkolů.
* **Pro komunikaci:** `@Email.md` vám pomůže udržet profesionální tón.

## 📁 Seznam dostupných agentů a rolí
* **Webova-stranka-UX.md**: Role seniorního UX designéra pro letectví.
* **PPC-zadani.md**: Standardizovaná struktura zadání pro specialisty PPC.
* **ClickUp-task.md**: Definice, jak má vypadat úkol pro tým.
* **Externi-grafika.md**: Role Art Directora pro úkolování grafiků.
* **Email.md**: Šablony a tón komunikace.

## 🛠 Propojené systémy (Context)
Při používání těchto promptů Cursor automaticky zohledňuje:
* **CMS:** Upgates
* **CRM:** HubSpot
* **PM:** ClickUp
* **ERP:** Pohoda
