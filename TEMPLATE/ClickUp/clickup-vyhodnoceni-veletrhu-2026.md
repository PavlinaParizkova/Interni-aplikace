Vyplňuje se v **ClickUp** – složka Eventy, příslušný task. Vyplňte do 1 týdne po veletrhu. Spolupracujte s obchodem – data o schůzkách a leadech.

### 2. Top 5 klíčových obchodních jednání

Pro každé jednání vyplňte: **Jméno firmy**, **Země**, **Popis firmy a první dojem** (typ firmy, zaměření, první setkání), **Klíčové téma/poptávka**, **Současný stav obchodního případu**, **Výsledek jednání**, **Potenciální hodnota a pravděpodobnost** (např. 10 000 000 CZK s pravděpodobností 50 %), **Plánované kroky po veletrhu** (např. „Zaslat nabídku do 14 dnů“, „Domluvit schůzku v kanceláři“).

#### 3A. Vyhodnocení odměny – nesales tým (jednoduchý model)

Vyplňte pouze pro nesales role (marketing, backoffice, logistika, produkt, technická podpora).


| Jméno | Role | KPI 1 Příprava (0-25) | KPI 2 Výkon na místě (0-35) | KPI 3 Follow-up (0-25) | Týmový cíl (0-15) | Body celkem (0-100) | Bonus (Kč) | Důkazy (max. 3 odkazy) |
| ----- | ---- | --------------------- | --------------------------- | ---------------------- | ----------------- | ------------------- | ---------- | ---------------------- |
|       |      |                       |                             |                        |                   |                     |            |                        |


**Bonusová pásma:**

- méně než 60 b. = 0 Kč
- 60-74 b. = 2 000 Kč
- 75-89 b. = 3 000 Kč
- 90-100 b. = 5 000 Kč

**Schválení:** event lead + přímý manažer.  
**Termín vyhodnocení:** do 7 dnů po veletrhu.
