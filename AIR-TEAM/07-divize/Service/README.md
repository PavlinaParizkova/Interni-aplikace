# Service Division Hub

Centrální dlouhodobá dokumentace entity `Service`, která zastřešuje servisní, opravárenské a modernizační kompetence `AIR TEAM`. Tento hub je závazným zdrojem pravdy pro roli servisní divize, její služby a způsob, jak má komunikovat k provozovatelům letadel a technickým zákazníkům.

## Co je uvnitř

- `00-strategie/master-division-strategy.md` - hlavní sjednocený strategický dokument entity

## Jak s hubem pracovat

- Začněte vždy v `00-strategie/master-division-strategy.md`.
- Před přípravou jakéhokoli materiálu si určete, pro koho je určen a jakou míru technické hloubky a jistoty komunikace potřebuje.
- Všechny servisní, obchodní a prezentační materiály musí vycházet z této strategie.
- Tento hub je závazný pro komunikaci entity `Service` a nesmí být svévolně oslabován nebo přepisován.

## Doporučené budoucí rozšíření

- `01-messaging/` - messaging framework pro servis, údržbu a modernizace
- `02-visual-guidelines/` - pravidla pro technické a prezentační materiály
- `03-content-playbooks/` - playbooky pro web, reference, případové studie a servisní obsah
- `04-campaign-templates/` - šablony pro nabídky, poptávky a servisní prezentace
