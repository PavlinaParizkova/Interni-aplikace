# JetConcept Strategy Hub

`Strategie: JetConcept`

Centrální dlouhodobá dokumentace entity `JetConcept`, která slouží jako strategický základ pro integrační, certifikační a programovou část portfolia `AIR TEAM`. Hub nově pokrývá jak hlavní positioning entity, tak detailnější servisně-technickou vrstvu pro `DOA`, `POA`, engineering a přestavby letadel.

## Co je uvnitř

- `00-strategie/master-division-strategy.md` - hlavní strategický dokument entity
- `01-services-strategy/jetConcept-strategy.md` - samostatná servisně-technická strategie pro `DOA`, `POA`, completions a conversions

## Jak s hubem pracovat

- Začněte vždy v `00-strategie/master-division-strategy.md`.
- Pokud připravujete capability materiál, nabídku nebo obchodní argumentaci, pokračujte do `01-services-strategy/jetConcept-strategy.md`.
- Před přípravou jakéhokoli materiálu si určete, pro který segment je určen: `business` nebo `commercial`.
- Komunikace má být věcná, autoritativní, klidná a důvěryhodná.

## Doporučené budoucí rozšíření

- `02-messaging/` - messaging framework podle segmentů a typů projektů
- `03-content-playbooks/` - playbooky pro capability decky, reference a webový obsah
- `04-campaign-templates/` - šablony pro nabídky, partnerské materiály a obchodní follow-up
- `05-go-to-market/` - navazující marketingové a veletržní strategie, včetně `AIX`
