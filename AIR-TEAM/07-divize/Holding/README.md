# AIR TEAM Holding Hub

Centrální dlouhodobá dokumentace entity `AIR TEAM Holding`, která slouží jako zastřešující strategický hub celé skupiny. Tento hub je závazným zdrojem pravdy pro korporátní směr, postavení holdingu a způsob, jak má holding mluvit o portfoliu divizí a služeb.

## Co je uvnitř

- `00-strategie/master-division-strategy.md` - hlavní sjednocený strategický dokument holdingu

## Jak s hubem pracovat

- Začněte vždy v `00-strategie/master-division-strategy.md`.
- Před přípravou jakéhokoli materiálu si určete, komu je určen a jakým tónem má holding mluvit.
- Všechny návazné dokumenty musí respektovat roli holdingu jako strategického lídra a integrátora portfolia.
- Tento hub je závazný pro korporátní prezentaci `AIR TEAM` a nesmí být svévolně oslabován nebo přepisován.

## Doporučené budoucí rozšíření

- `01-messaging/` - korporátní messaging framework podle publika
- `02-visual-guidelines/` - pravidla pro holdingovou prezentaci a portfolio architekturu
- `03-content-playbooks/` - playbooky pro partnerskou, náborovou a korporátní komunikaci
- `04-campaign-templates/` - šablony pro prezentace, nabídky a strategické materiály
