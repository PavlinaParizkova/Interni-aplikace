# Intel Division Hub

Centrální dlouhodobá dokumentace entity `Intel`, která zastřešuje obranná, speciální a technologicky náročná řešení `AIR TEAM`. Tento hub je závazným zdrojem pravdy pro roli entity, její cílové skupiny a způsob, jak má mluvit v prostředí kritických misí a specializovaných projektů.

## Co je uvnitř

- `00-strategie/master-division-strategy.md` - hlavní sjednocený strategický dokument entity

## Jak s hubem pracovat

- Začněte vždy v `00-strategie/master-division-strategy.md`.
- Před přípravou jakéhokoli materiálu si určete, jestli komunikujete k obrannému, vládnímu, nebo firemnímu publiku, a upravte tomu úroveň detailu i tón.
- Všechny prezentační, obchodní a obsahové výstupy musí vycházet z této strategie.
- Tento hub je závazný pro komunikaci entity `Intel` a nesmí být svévolně rozmělňován nebo přepisován.

## Doporučené budoucí rozšíření

- `01-messaging/` - messaging framework pro B2G a specializované B2B publikum
- `02-visual-guidelines/` - pravidla pro diskrétní a profesionální prezentaci
- `03-content-playbooks/` - playbooky pro case studies, capability decky a technický obsah
- `04-campaign-templates/` - šablony pro nabídky, capability prezentace a jednání
