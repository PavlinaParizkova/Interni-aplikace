## Spouštěcí checklist (před zahájením produkce)

- [ ] Název kampaně a značka / klient potvrzen
- [ ] Nabídka a podmínky jsou schváleny (přesný text akce)
- [ ] Cílová skupina a trhy jsou jasně definovány
- [ ] Kanály jsou potvrzeny (email, social, PPC, web banner, landing page)
- [ ] Rozpočet po kanálech schválen Marketing managerem
- [ ] KPI cíle (primární + sekundární) jsou zapsané v ClickUp tasku
- [ ] UTM naming convention je připraven
- [ ] Owner a schvalovatel jsou přiřazeni

**Bez splnění tohoto checklistu task nepřechází z `Brief` do `Planning`.**

---

## Status workflow v ClickUp

```
Brief → Planning → Production → QA → Ready to Launch → Live → Reporting → Closed
```

## Klíčové deadliny

| Fáze | Deadline |
|------|----------|
| Brief schválen | T-14 |
| Assety (copy + grafika) schváleny | T-5 |
| QA checklist 100 % | T-2 |
| Launch | T0 |
| Průběžná optimalizace | T+1 až T+X |
| Vyhodnocení | do 5 pracovních dnů po konci |

---

## QA checklist před spuštěním

- [ ] Nabídka a podmínky jsou konzistentní ve všech kanálech
- [ ] Všechny proklikávací odkazy mají UTM parametry
- [ ] Hero sekce a CTA jsou viditelné na mobilu bez scrollu
- [ ] Test v Gmailu schválen
- [ ] Produktové obrázky jsou prolinkované
- [ ] HubSpot campaign je přiřazená
- [ ] Jazykové mutace jsou zkontrolované
- [ ] Schvalovatel dal finální Go

---

## Launch checklist (T0)

1. Spustit v plánovaný čas (typicky 18:00 nebo dle segmentu).
2. Po spuštění čekej 5 minut, zkontroluj doručení a živé URL.
3. Prvních 60 minut monitoruj:
   - [ ] E-mail doručen (HubSpot dashboard)
   - [ ] Všechny proklikávací URL jsou funkční
   - [ ] E-shop / landing page nevede na 404
   - [ ] GA4 event fires (konverzní tracking aktivní)
4. Pokud je chyba: **Nevypínat kampaň bez konzultace s Marketing managerem.**

---

## Uzavření tasku

Task lze přepnout na **Closed** pouze pokud:
- [ ] Sekce **Vyhodnocení** je vyplněna v ClickUp tasku
- [ ] **Lessons learned** jsou zapsány (min. 3 body)
- [ ] Min. 3 navazující akční kroky s ownerem a deadlinem jsou zadány
