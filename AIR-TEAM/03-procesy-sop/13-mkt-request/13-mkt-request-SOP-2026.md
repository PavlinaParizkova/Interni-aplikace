## Pro zadavatele: Jak podat MKT Request

### Krok 1: Otevři ClickUp formulář

1. Přejdi do ClickUp → seznam **MKT Request**.
2. Klikni na **Form view** nebo použij sdílený formulářový odkaz.
3. Vyplň všechna povinná pole.

### Povinná pole v MKT Request formuláři

| Pole | Popis |
|------|-------|
| **Co potřebuješ?** | Grafika / text / web update / post / kampaň / jiné |
| **Proč to potřebuješ?** | Kontext – k čemu to slouží |
| **Pro koho je výstup?** | Zákazník / partner / tým / média |
| **Deadline** | Datum, kdy nejpozději potřebuješ |
| **Podklady** | Odkaz na materiály, texty, reference, logo (vlož do přílohy nebo Google Drive) |
| **Schvalovatel** | Kdo schválí výstup z tvé strany před použitím |

### Krok 2: Odeslání a potvrzení

- Po odeslání formuláře dostaneš automatické potvrzení e-mailem.
- Marketing tým se ozve do **1 pracovního dne** s potvrzením přijetí a odhadovaným termínem.
- Pokud máš urgenci: označ task jako Urgentní a zavolej Marketing coordinátorovi.

## Pro marketing tým: Zpracování MKT Request

### Krok 1: Příjem a třídění (každé ráno)

1. Otevři ClickUp → MKT Request → nové úkoly (stav: **Inbox**).
2. Pro každý nový request:
   - [ ] Zkontroluj, zda jsou vyplněna všechna povinná pole
   - [ ] Pokud ne: přepni na stav **Waiting for Info** a napiš zadavateli co chybí
   - [ ] Pokud ano: přejdi na krok 2

### Krok 2: Prioritizace

Přiřaď prioritu dle SLA tabulky (viz `13-mkt-request-PROCES-2026.md`):

- Je deadline < 24 hodin? → Urgentní → okamžitě informovat Marketing managera
- Je deadline < 5 pracovních dní? → Vysoká priorita
- Je deadline < 10 pracovních dní? → Standardní
- Bez deadlinu nebo >10 dní → Nízká

### Krok 3: Přidělení specialistovi

1. Přepni task na stav **In Review**.
2. Přiřaď zodpovědného specialistu:
   - Grafika → Grafický designér
   - Text / post → Content manager
   - Web → Digital specialist
   - Kampaň / komplex → Marketing manager přidělí

3. Nastav deadline v ClickUp (dle SLA).
4. Odešli zadavateli potvrzení:

```
Ahoj [jméno],

tvůj MKT Request jsme přijali.

Co: [stručný popis]
Zodpovídá: [jméno specialisty]
Odhadovaný termín dodání: [datum]

Pokud se termín změní, dáme ti vědět.
```

### Krok 4: Realizace

Specialista realizuje dle příslušného SOP:
- Grafika → viz `09-grafika-foto-video-SOP-2026.md`
- Blogy / text → viz `03-blogy-clanky-SOP-2026.md`
- Web update → viz `02-web-landing-SOP-2026.md`
- Social post → viz `01-socialni-site-SOP-2026.md`

### Krok 5: Dodání a uzavření

1. Specialista sdílí výstup s zadavatelem (Google Drive odkaz nebo příloha v ClickUp).
2. Přepni task na stav **Delivered**.
3. Informuj zadavatele:

```
Ahoj [jméno],

tvůj MKT Request je hotový.

Výstup: [odkaz nebo popis]
Pokud jsou potřeba drobné úpravy, napiš do 2 pracovních dnů.
```

4. Po potvrzení zadavatele: přepni task na **Done**.

## Eskalace a výjimky

| Situace | Řešení |
|---------|--------|
| Zadavatel požaduje kratší deadline než SLA | Konzultovat s Marketing managerem; buď přijmeme nebo přesměrujeme zákazníka |
| Request je mimo scope marketingu | Odmítnout s vysvětlením a navrhnutím jiného řešení |
| Zadavatel neposkytl podklady ani po připomínce | Po 3 pracovních dnech bez odpovědi archivovat task s komentářem |
| Urgentní request mimo pracovní dobu | Vyřizujeme pouze pokud je to schváleno Marketing managerem (definice opravdové urgence) |
