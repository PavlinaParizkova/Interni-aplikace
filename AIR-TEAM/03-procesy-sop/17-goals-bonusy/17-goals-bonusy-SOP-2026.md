## Část A: Nastavení cílů (kvartál nebo rok)

### Krok 1: Příprava cílů Marketing managerem

1. Prostuduj výsledky předchozího období (viz `16-reporting-SOP-2026.md`).
2. Pročti firemní OKR a plán vedení na Q (nebo rok).
3. Navrhni KPI pro každého člena týmu a pro tým jako celek.

### Krok 2: Formát KPI pro každého člena týmu

```
Jméno: [jméno]
Pozice: [pozice]
Kvartál/Rok: [Q/rok]

## Osobní KPI (zodpovídám sám/a)

| KPI | Cílová hodnota | Zdroj dat | Váha |
|-----|----------------|-----------|------|
| [metrika] | [číslo/procento] | [HubSpot/GA4/ClickUp] | [%] |
| ... | | | |
Celkem vah: 100 %

## Týmové KPI (spoluzodpovídám)

| KPI | Cílová hodnota | Zdroj dat |
|-----|----------------|-----------|
| [metrika] | | |

## Rozvoj / projekty (pokud relevantní)
- [cíl / projekt / certifikace]
```

### Krok 3: Konzultace s členem týmu

1. Sdílej návrh KPI s každým členem týmu.
2. Proběhne schůzka (15–30 minut):
   - Vysvětli kontext a propojení s firemní strategií
   - Získej zpětnou vazbu a doplň případné návrhy člena týmu
   - Dohodněte se na finálním znění
3. Finální KPI oběma stranami odsouhlasit (e-mail nebo podpis v dokumentu).

### Krok 4: Záznam v ClickUp

1. Založ task v ClickUp → Goals & Bonusy → `[jméno] – Q[číslo] [rok]`.
2. Přidej jako checklist nebo description finální KPI.
3. Nastav deadline na konec Q (nebo roku).

---

## Část B: Průběžné sledování (měsíční check-in)

### Každý 1. týden v měsíci

1. Každý člen týmu aktualizuje svůj KPI task v ClickUp:
   - Aktuální hodnota metrik vs. cílová hodnota
   - Komentář: na cestě / riziko / blocker
2. Marketing manager projde všechny tasky (15 minut).
3. Pokud je plnění pod 60 % a zbývají méně než 2 měsíce do konce Q → schůzka a akční plán.

---

## Část C: Vyhodnocení (poslední týden Q nebo roku)

### Krok 1: Sběr finálních dat

Pro každý KPI exportuj nebo ověř hodnoty ze zdrojů dat (HubSpot, GA4, ClickUp).

### Krok 2: Výpočet plnění

Pro každé KPI: `Plnění (%) = Skutečnost / Cíl × 100`

Příklad výsledkové tabulky:

```
Jméno: [jméno]
Kvartál: [Q]

| KPI | Cíl | Skutečnost | % plnění | Váha | Vážený výsledek |
|-----|-----|-----------|----------|------|-----------------|
| ... | | | | 40 % | |
| ... | | | | 30 % | |
| ... | | | | 30 % | |
| **CELKEM** | | | | 100 % | **X %** |
```

### Krok 3: Doporučení na bonus

Bonusová pásma (nastavit dle interní HR politiky):

| Plnění | Doporučení |
|--------|------------|
| ≥ 100 % | Plný bonus |
| 90–99 % | Bonus dle dohody |
| 75–89 % | Částečný bonus |
| < 75 % | Bez bonusu |

Doporučení zaznamenej do Google Sheets šablony pro HR.

### Krok 4: Schválení a komunikace

1. Výsledky sdílej s každým členem týmu individuálně (1:1 meeting).
2. Pošli finální přehled HR / Finance ke zpracování výplaty.
3. Dokumentaci ulož do ClickUp tasku a Google Drive (HR složka).

---

## Část D: Roční personal goals

### Postup nastavení (říjen–listopad)

1. Každý člen týmu připraví návrh osobních cílů na příští rok (volný formát, max. 5 bodů).
2. Marketing manager doplní firemní kontext.
3. Společná schůzka (30 minut) → dohoda na finálním znění.
4. Záznam v ClickUp a podpis (e-mail nebo dokument).

### Typy personal goals
- Kompetence: nová dovednost, kurz, certifikace
- Projekt: vedení specifického projektu nebo iniciativy
- Leadership: mentorování kolegy, vedení schůzek

## Šablona e-mailu: oznámení výsledků

```
Předmět: Vyhodnocení KPI – [jméno] – Q[číslo] [rok]

Ahoj [jméno],

přikládám výsledky tvého hodnocení za Q[číslo]:

Celkové plnění: [X] %
Bonus: [popis nebo Ano/Ne dle pásma]

Detaily: [odkaz na dokument nebo tabulku]

Děkuji za práci v tomto kvartálu.
Příští Q cíle probereme [datum/čas schůzky].
```

---

## Krizové scénáře

| Problém | Řešení |
|---------|--------|
| Člen týmu nesouhlasí s hodnocením | 1:1 schůzka, přezkoumání dat; pokud shoda nenastane – eskalovat na HR |
| KPI data nejsou dostupná (technický problém) | Odhadnout na základě alternativních zdrojů; zdokumentovat |
| Cíle jsou nereálné (zjistí se v průběhu) | Revize na měsíčním check-inu; aktualizovat po dohodě obou stran |
