# AEROSPEC Brand Hub

Centrální dlouhodobá dokumentace značky `AEROSPEC`, která slouží jako samostatná brandová větev mimo kampaně. `AEROSPEC` je značka `AIR TEAM` a tento hub má fungovat jako závazný zdroj pravdy pro budoucí strategické, obsahové, produktové i exekuční materiály.

## Co je uvnitř

- `00-strategie/master-brand-strategy.md` - hlavní sjednocený dokument značky
- `01-messaging/core-messaging-framework.md` - rozpracovaný messaging framework značky
- `03-content-playbooks/tone-of-voice-prispevky-a-brozury.md` - tone-of-voice playbook pro obsah a prezentační materiály

## Jak s hubem pracovat

- Začněte vždy v `00-strategie/master-brand-strategy.md`.
- Kampaňové dokumenty vytvářejte až jako navazující vrstvu nad touto strategií.
- Při rozšiřování zachovávejte logiku: nejdřív strategie, potom messaging, vizuální pravidla, obsahové playbooky a kampaně.
- Tento hub je závazný pro celou značku a nesmí se svévolně upravovat.
- Jakákoli změna bez výslovného strategického rozhodnutí ohrožuje konzistenci značky `AEROSPEC`.

## Doporučené budoucí rozšíření

- `01-messaging/` - rozpracovaný messaging framework a claimy podle kanálů
- `02-visual-guidelines/` - detailní pravidla pro vizuální aplikaci značky
- `03-content-playbooks/` - obsahové a tone-of-voice playbooky
- `04-campaign-templates/` - šablony pro budoucí kampaně AEROSPEC
